基本类型

类型传值方式能修改原数据?
int, float, string拷贝值需要传指针
array拷贝整个数组需要传指针
struct拷贝结构体需要传指针
pointer拷贝地址通过指针修改
slice拷贝头部可修改元素，append 不影响
map拷贝指针可以
channel拷贝引用可以

为什么 slice 的 append 不影响?

slice 包括三部分:指针,长度,容量

传参时拷贝了这三部分，函数内 append 不会影响这三部分

func f(p []int) {
	p = append(p, 1)
}

func main() {
	var q = []int{1, 2, 3}
	f(q)
	fmt.Println(len(q))
	fmt.Println(cap(q))
}
